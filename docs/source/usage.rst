Usage
=====

Overview
--------

Below are suggestions on how you might like to configure the app for different use scenarios.
|app| is highly flexible — you can set up multiple :doc:`workspaces` for different study contexts
and switch between them easily.

Bible Study
-----------

Open a Bible in one window and a commentary in another using :doc:`windows`.
Enable synchronized scrolling so the commentary follows your Bible reading.
Use :ref:`cross-references` to discover related passages and
add :doc:`bookmarks` with :doc:`notes` as you study.

Topical Study
-------------

Create a dedicated :doc:`workspace <workspaces>` for your topic. Use :doc:`study_pads`
to collect verses and notes on a specific theme. The :doc:`search` feature can help
you find all occurrences of key words across your installed Bible modules.

Word Study
----------

Install a Strong's dictionary and enable Strong's number display in the toolbar.
Tap any underlined word to see its original language definition. Use the
:doc:`links window <documents>` to keep the dictionary visible alongside your Bible text.

Personal Devotions
------------------

Set up a :doc:`reading plan <reading_plans>` and use the :doc:`speak` feature to listen
to the text being read aloud. Add personal :doc:`notes` to verses that stand out during
your reading.

Sermon notes
------------

Use a :doc:`study pad <study_pads>` to organize your sermon outline. You can add verses
from different books and write rich-text notes between them. :doc:`Labels <labels>` can
help you categorize your sermon material by topic or series.

Original Language Study
-----------------------

Install Strong's Greek and Hebrew dictionaries and a morphology dictionary (like Robinson's).
Configure them in Application preferences. Enable Strong's number display in the toolbar
to see the original language codes. Tap words to see definitions and parsing information.
Use multiple :doc:`windows` to compare translations side by side.
