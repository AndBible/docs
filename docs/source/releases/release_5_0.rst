Release 5.0
===========

Release Date: **2023**

Major new features and improvements.
------------------------------------

* **ePub document support:** Digital books in ePub format can now be loaded and read as generic books.
* **Cloud synchronization:** Sync your bookmarks, notes, and settings across devices via Google Drive.
* **Discrete mode:** A separate "calculator" disguised app and a discrete mode in the main app for use in persecution-sensitive areas.
* **Non-Bible bookmarks and notes:** Bookmarks, highlighting, and notes are now supported in commentaries and other non-Bible documents.
* **Custom repositories:** Add third-party SWORD module repositories with JSON manifest support.
* **Chained links windows:** Links windows can now be chained, allowing you to follow references through multiple linked windows.
* **Improved Text-to-Speech:** Better TTS support for non-Bible documents.
* **Selective restore:** Choose exactly what to restore when importing a backup — no longer all-or-nothing.
* **Deep linking:** Open specific Bible references via ``https://read.andbible.org/`` URLs.
* **MyBible and MySword module support:** Experimental support for importing MyBible and MySword format modules.
* **Workspace colors:** Assign colors to workspaces for easier identification.
* **Label search and filtering:** Search and filter your labels for faster access.
* **TypeScript migration:** Internal improvement — BibleView.js migrated from JavaScript to TypeScript for better maintainability.

Many more bug fixes, features, and improvements.
